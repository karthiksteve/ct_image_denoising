# Instructions for Adding Presentation

This folder is for your PowerPoint presentation slides.

## Expected Content
- Project overview and motivation
- Architecture diagrams (Dual-Head UNet, PatchGAN, Global Discriminator)
- Training methodology and loss functions
- Results: PSNR/SSIM metrics, visual comparisons
- Conclusion and future work

## File Format
- `.pptx` (PowerPoint)
- Filename suggestion: `GAN_CT_Denoising_Presentation.pptx`

## Tips
- Include diagrams from `../results/` folder
- Show before/after comparison images
- Highlight key metrics from training
